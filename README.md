# pigGame
 É um jogo multiplayer 1v1 com uma dinâmica interessante de assumir riscos com o objetivo final em acumular 100 pontos primeiro que o adversário. Esse projeto teve seu HTML e CSS criado por Jonas Schmedtmann e teve sua lógica JS desenvolvida por mim. Nesse projeto me desafiei a criar toda a lógica sozinha antes de ver a solução no curso com o objetivo de aprender a resolver problemas sozinho e melhorar o meu pensamento lógico.
